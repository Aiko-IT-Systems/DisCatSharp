# API Reference

Welcome to the DisCatSharp.Hosting.AspNetCore API reference.

To begin, select a namespace, then a class, from the table of contents on the left.

The ASP.NET Core hosting package also exposes helper APIs for:
- computing the public ingress URLs for OAuth, interactions, and webhook endpoints,
- generating starter proxy snippets for NGINX, Apache, and common Docker reverse proxies,
- validating local ingress options against Discord application metadata such as redirect URIs, interactions URLs, verify keys, and role-connections verification URLs.
