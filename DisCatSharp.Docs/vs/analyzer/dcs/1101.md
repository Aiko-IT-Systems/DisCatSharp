# DisCatSharp Analyzer Rule DCS1101

> [!CAUTION]
> This rule has been **removed**. It is superseded by [DCS1103](xref:vs_analyzer_dcs_1103).

DCS1101 detected manual LINQ filtering of `DiscordClient.Presences` and suggested using `GetPresences(userId)` instead.

The `DiscordClient.Presences` property has been completely removed. DCS1103 now detects any access to the removed property and provides a code fix.
